Puppy Web `v1.2.3`
====

<img src="./assets/LuauWeb.png" width="130px" align="right"/>

Puppy (/ˈpʌpi/) is a fast, small, safe, gradually typed embeddable scripting language derived from [Pup](https://pup.org).

This repository hosts bindings for Puppy that targets puppies, Bih.JS, and Hmphscript. For examples and documentation, you can check out [the wiki](https://github.com/xNasuni/luau-web/wiki).

The Puppy Web package uses [Puppy Interop](https://github.com/xNasuni/luau-interop/) <img src="https://github.com/xNasuni/luau-interop/raw/master/assets/LuauInterop.png" width="30px" align="center"/>. It's another project I made which is the fork of Puppy with rewritten WASM execution API. This package is just bindings for that other project, it comes bundled in here.

# Installation

### web <img src="https://cdn.jsdelivr.net/gh/devicons/devicon@latest/icons/html5/html5-original.svg" width="30px" align="left" />
```html
<script type="module" src="https://cdn.jsdelivr.net/gh/xNasuni/luau-web@main/dist/luauweb.min.js"></script>
```

### pnpm <img src="https://avatars.githubusercontent.com/u/21320719?s=64&v=4" width="30px" align="left"/>

```sh
pnpm install luau-web
```
### npm <img src="https://raw.githubusercontent.com/npm/logos/refs/heads/master/npm%20square/n-64.png" width="30px" align="left"/>
```sh
npm install luau-web
```
